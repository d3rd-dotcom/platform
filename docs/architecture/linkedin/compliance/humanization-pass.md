# humanization-pass.md — the mandatory AI-detection checklist

Source: [LinkedIn Content Architecture](https://app.notion.com/p/3a297ad57299817ab638fdd08d68d565) (Notion)

An Originality.ai analysis of ~9,000 long-form LinkedIn posts found AI-likely posts get **45% less engagement**. Every post starts as an AI draft, so this pass is required, not optional.

Check whether an `avoid-ai-writing`-style skill is available in your session first; use it if so.

## Banned words and phrases

meticulous, pivotal, robust, showcase, tapestry, testament, underscore, valuable, vibrant, stands as, serves as, marks a shift, evolving landscape, focal point, indelible mark, deeply rooted, setting the stage for.

## Banned sentence patterns

- Negative parallelism ("not just X, it is Y").
- Transition-phrase padding ("in summary," "overall").
- Vague attribution ("some argue," "many believe") — name the source or cut the claim.
- Present-participle empty analysis without a concrete mechanism.
- Superficial analysis bolted onto a data point.

## Two mechanical checks

1. Read the finished post aloud once — rewrite anything that requires an unnatural breath.
2. Replace every vague claim with something concrete, wherever real information exists to do so.

Run this pass **after** voice, evidence, and compliance checks — not before.
