Descriptive statistics summarize the data you collected. They help a reader see
the center, spread, and shape of a set of observations before anyone tries to
generalize beyond it.

**Center**

The mean is the sum of all values divided by the number of values. The median is
the middle value after sorting. The mode is the value that appears most often.
For the set 2, 3, 3, 4, 8, the mean is 4, the median is 3, and the mode is 3.
The high value pulls the mean upward, which is why reporting more than one
summary can help.

**Spread**

The range is the largest value minus the smallest. A data set can have the same
mean as another set while varying much more. A measure of spread gives readers
context for the center. A chart can reveal clusters, gaps, and unusual values
that one number hides.

**Describe before you explain**

If a class records focus ratings before and after a quiet study period, first
report the number of observations and the summaries for each time. Then describe
the visible pattern. Avoid saying the quiet period caused a change unless the
design supports that conclusion. A descriptive summary tells what happened in
the recorded data. Explaining why it happened or whether the same pattern would
appear in another group requires additional evidence.

For a small project, show the raw values, the calculation, and a simple chart.
Label the units and the question. Clear arithmetic makes a result easier to
check, discuss, and improve.
