Without memory, everyday tasks fall apart — you wouldn't know how to dress yourself, tie your shoes, recognize family, find food, or cook a meal. This chapter looks at how memory is stored and retrieved, why we forget, and — in the section below — concrete strategies for improving recall and studying more effectively.

**Memory-enhancing strategies**

You can move information from short-term to long-term memory using a few core techniques:

- Rehearsal: consciously repeating information, like learning multiplication tables through repetition.
- Chunking: organizing information into manageable pieces — remembering "520-555-0467" instead of "5205550467."
- Elaborative rehearsal: thinking about meaning and connections, like noting that 520 is Arizona's area code to help remember a phone number from that state.
- Mnemonic devices: for larger units of information, like "My Very Educated Mother Just Served Us Nachos" for remembering the order of the planets.

Two additional findings: expressive writing about traumatic experiences for 20-minute sessions boosted short-term memory capacity after five weeks in one study, and speaking words aloud increases their distinctiveness, improving memory retention by more than 10%.

**How to study effectively**

- Use elaborative rehearsal and deeper processing rather than surface-level review.
- Apply the self-reference effect — connect material to something personally meaningful.
- Use distributed practice spread across time rather than cramming.
- Rehearse material repeatedly in organized sessions.
- Use index cards instead of highlighting.
- Reduce interference by studying in a quiet environment.
- Exercise regularly, which promotes neurogenesis.
- Get adequate sleep for memory consolidation.
- Use mnemonic devices like acronyms and acrostics.
